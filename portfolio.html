Album management CRUD application

This is a simple CRUD application for managing a list of albums, including the album name, artist or band name, genre and release year. I've loaded some random albums that I love here. Feel free to add your own, correct any mistakes you see on those that are currently showing or delete any albums you're simply not crazy about.

The backend is written in Java and Spring using the MVC design pattern. It is deployed via Elastic Beanstalk and uses DynamoDB as the database. The frontend is written in React using Axios as the HTTP client. The source code is available here.
